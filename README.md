## Titanic Survival Prediction

A machine learning project that analyzes and predicts passenger 
survival on the Titanic using Logistic Regression and Decision Tree models.

## Dataset
- Source: Stanford University (titanic.csv)
- 887 passengers, 8 features
- No missing values

## Analysis Highlights
- Overall survival rate: 38.56%
- Female survival rate: 74.20%
- Male survival rate: 19.02%
- 1st class survival rate: 62.96%
- 3rd class survival rate: 24.44%

## Models & Results
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 76.40% |
| Decision Tree | 79.78% |

Best model: Decision Tree with 79.78% accuracy

## Features Used
- Passenger Class (Pclass)
- Gender (Sex)
- Age
- Fare
- Family Size
- Is Alone

## Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
