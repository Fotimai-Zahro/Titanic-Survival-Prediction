# Titanic Survival Prediction

A machine learning project that predicts whether a passenger 
survived the Titanic disaster based on features like age, 
gender, and passenger class.

## Technologies Used
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

## Project Structure
- Data exploration and visualization (EDA)
- Data preprocessing and cleaning
- Model training and evaluation
- Survival predictions
